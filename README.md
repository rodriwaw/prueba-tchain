# prueba-tchain
Prueba técnica TChain

## Cómo correr ambos proyectos

### Backend

1. Navega al directorio del backend:
    ```sh
    cd /ruta/al/backend
    ```
2. Instala las dependencias:
    ```sh
    npm install
    ```
3. Inicia el servidor:
    ```sh
    npm start:prod
    ```

### Frontend

1. Navega al directorio del frontend:
    ```sh
    cd /ruta/al/frontend
    ```
2. Instala las dependencias:
    ```sh
    npm install
    ```
3. Inicia la aplicación:
    ```sh
    npm start:prod
    ```
